# CinPressa Pharma

Landing splash page for **CinPressa Pharma**, an asset-focused CinCo within the [CinRx Pharma](https://cinrx.com) portfolio.

## Stack

- Next.js (App Router, RSC)
- Tailwind CSS v4
- Montserrat (Gotham stand-in per the brand spec sheet)

## Brand

Colors and the mark follow the official CINPRESSA Logo Spec Sheet (locked, never reinterpret):

- Core blue `#2261AD` · Sky (wordmark) `#3AAED8` · Orange `#F9A81A` (punctuation only)
- Mark: full-color Illustrator recreation (`public/cinpressa-mark.svg`). No red, no outline versions.

## Develop

```bash
npm install
npm run dev
```

Deployed on Vercel.
